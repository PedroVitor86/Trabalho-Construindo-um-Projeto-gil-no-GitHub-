# Kanban - GitHub Projects (Sugestão de cards)
Colunas: A Fazer | Em Progresso | Concluído

Exemplos de cards (mínimo 10):
- [A Fazer] Criar scaffold do projeto (README, estrutura)
- [A Fazer] Implementar CRUD básico (Flask)
- [A Fazer] Configurar DB SQLite
- [A Fazer] Escrever testes unitários (pytest)
- [A Fazer] Configurar GitHub Actions (CI)
- [Em Progresso] Implementar interface HTML básica
- [Em Progresso] Fazer integração dos templates
- [Concluído] Inicializar repositório
- [Concluído] Documentar instruções de execução
- [A Fazer] Simular mudança de escopo (autenticação)
Dica: Use labels (bug, enhancement, docs), assignees e milestones para organizar.
